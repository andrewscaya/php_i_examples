Session Save Handlers:
1. Depends on installation; use phpinfo() to find list for your server
2. Default = "files"
   This is documented here: http://php.net/manual/en/function.session-set-save-handler.php
3. You can write your own using the SessionHandler class
   see: http://php.net/manual/en/class.sessionhandler.php
4. There are some alternatives to "files" listed in pecl.php.net
   memsession:    http://pecl.php.net/package/memsession
   session_mysql: http://pecl.php.net/package/session_mysql
